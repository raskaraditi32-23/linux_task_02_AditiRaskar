# Linux Task 02 - Users, Groups & File Permissions


# Objective

This task focuses on learning Linux user management, groups, file ownership, and permissions. It helps build a strong foundation in Linux security and system administration.

# Topics Covered

- Linux Users
- Linux Groups
- UID & GID
- /etc/passwd
- File Ownership
- chmod Command
- chown Command
- Numeric Permissions
- Linux Security
- Principle of Least Privilege



## Outcome

Successfully learned Linux user management, group management, file ownership, permission settings, and Linux security best practices.
